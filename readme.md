本项目是TGAM脑电模块-疲劳驾驶检测一个基于Spring Boot + Vue.js构建的一个前后端分离项目框架的后端项目代码部分。  
## 后端技术栈
+ SpringBoot   
+ Mybatis plus  
+ Swagger2-ui+knife4j
+ aliyun-sdk-oss
+ jwt
